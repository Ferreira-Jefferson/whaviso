# business

Parte de negócio do whaviso: marketing, posicionamento, aquisição, conteúdo e materiais comerciais.

Esta pasta é independente do código (`backend/`, `frontend/`). Aqui não mora implementação, mora estratégia e os artefatos de negócio.

## Regras de linguagem (valem aqui também)

- Nunca usar "dívida", "devendo", "atraso", "cobrança", "inadimplência". Sempre "aviso/lembrete/combinado".
- Nunca usar travessão (em dash). Prefira vírgula, dois-pontos, parênteses ou reescreva a frase.

## Estrutura sugerida (crie conforme precisar)

- `marketing/`: campanhas, canais, copy, anúncios
- `posicionamento/`: proposta de valor, público, concorrência
- `conteudo/`: posts, artigos, roteiros
- `metricas/`: funil, aquisição, retenção
